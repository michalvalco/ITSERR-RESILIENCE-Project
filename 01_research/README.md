# 01_research

Research materials, sources, and literature notes for the ITSERR/RESILIENCE project.

## Structure

```
01_research/
├── sources/           # Markdown files for key sources
└── literature_notes/  # Annotated bibliographies
```

## Sources (`sources/`)

This folder contains markdown files summarizing key sources relevant to the project:

- **GNORM technical papers** — CRF-based annotation approaches
- **Digital humanities methodologies** — AI in religious studies
- **Hermeneutical theory** — Personalist philosophy, theological interpretation
- **AI ethics literature** — Responsible AI design principles

### Naming Convention

```
YYYY_AuthorLastName_ShortTitle.md
```

Example: `2025_Pavone_GNORM_CRF_Annotation.md`

## Literature Notes (`literature_notes/`)

Annotated bibliographies and reading notes organized by theme:

- `hermeneutics.md` — Notes on hermeneutical theory
- `ai_ethics.md` — AI ethics and responsible design
- `digital_humanities.md` — DH methodologies for religious studies
- `gnorm_itserr.md` — GNORM and ITSERR-specific materials

## Workflow

1. When encountering a relevant source, create a markdown file in `sources/`
2. Include: citation, abstract/summary, key quotes, relevance to project
3. Update the appropriate literature notes file with cross-references
4. Tag sources with relevant themes for easy retrieval
